# AutoVibe

AutoVibe is a car-focused social media mobile application.

## Features
- Social media feed for car content
- Reels-style video interface
- Car marketplace module
- Automotive news section
- User profiles

## Technologies
Frontend:
- Flutter

Backend:
- Node.js
- PostgreSQL

## Project Structure
autovibe-app/
 ├── frontend/   → Flutter mobile application
 ├── backend/    → Node.js backend server
 └── dev_run.bat → Development script
